### Console Game - Shooting Game

기간 : 2025.03.24 - 25.03.31

### 개요
콘솔 창에 작동되는 게임 만들기

### 목표
1. 게임 루프 사용하기
2. 콘솔창에 플레이어와 적 출력하기
3. 포인터, 할당 적응하기
4. 시간 기반 이동 사용하기
5. 파일 입출력 사용하기
6. FMOD를 사용해 소리 출력하기
